== Description ==

Blog Avatar Plugin allows site admins to upload avatars for their blog

= How it works:- =

Upload blog avatar from backend settings options.

= Help & Support =
Please leave a comment on our blog [BuddyDev](http://buddydev.com/buddypress/add-avatars-to-your-blogs-on-buddypresswordpress-multisite-network/) for any suggestions, support request.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.
1. Download the zip file and extract
1. Upload `blog-avatar` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu.
1. Enjoy

== Frequently Asked Questions ==

= Does This plugin works without BuddyPress =
No, It needs you to have BuddyPress Installed and activated
